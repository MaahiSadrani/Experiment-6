Q1) Algorithm to Check Whether a Number is Positive, Negative or Zero
Start
Input a number num
If num > 0, print "Number is positive"
Else if num < 0, print "Number is negative"
Else print "Number is zero"
Stop

Q2) Algorithm to Check Whether a Number is Even or Odd
Start
Input a number num
If num % 2 == 0, print "Number is even"
Else print "Number is odd"
Stop

 Q3) Algorithm to Find the Largest of Three Numbers
Start
Input three numbers a, b, and c
If a >= b and a >= c, then a is largest
Else if b >= a and b >= c, then b is largest
Else c is largest
Display the largest number
Stop

Q4) Algorithm to Calculate Grade
Start
Input marks
If marks ≥ 90 → Grade A
Else if marks ≥ 75 → Grade B
Else if marks ≥ 60 → Grade C
Else if marks ≥ 40 → Grade D
Else → Grade F
Display grade
Stop

 Q5) Algorithm to Check Leap Year
Start
Input year
If (year divisible by 4 AND not divisible by 100) OR (year divisible by 400)
Then print "Leap year"
Else print "Not a leap year"
Stop

Q6) Algorithm to Calculate Gross Salary
Start
Input basic salary
If basic ≤ 10000
HRA = 20% of basic
DA = 80% of basic
Else if basic ≤ 20000
HRA = 25% of basic
DA = 90% of basic
Else
HRA = 30% of basic
DA = 95% of basic
Gross Salary = Basic + HRA + DA
Display gross salary
Stop

Q7) Algorithm to Calculate Income Tax
Start
Input annual income
If income ≤ 2,50,000 → Tax = 0
Else if income ≤ 5,00,000
Tax = (income − 2,50,000) × 5%
Else if income ≤ 10,00,000
Tax = (2,50,000 × 5%) + (income − 5,00,000) × 20%
Else
Tax = (2,50,000 × 5%) + (5,00,000 × 20%) + (income − 10,00,000) × 30%
Display tax
Stop
